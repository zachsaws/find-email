- 📖 快速开始指南
- ⭐ 引导用户 Star 项目
- 🤝 社区贡献引导

## 🛠 技术栈

- HTML5 + CSS3 + JavaScript
- 响应式设计
- 现代化UI/UX
- SEO优化

## 🚀 部署

本网站使用 GitHub Pages 自动部署：
1. 文件放在 `docs/` 目录
2. GitHub 仓库设置中启用 GitHub Pages
3. 选择 `docs` 目录作为源
